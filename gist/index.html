Your files never leave your Mac.

Gist is built with Apple Intelligence. All processing — text extraction, image recognition, name generation — happens on your Mac using Apple's on-device frameworks. No internet connection. No cloud uploads. No API keys. No accounts. No telemetry.

Your bank statements, tax documents, medical records, and personal photos stay exactly where they are.

Heavy code. Light model.

Most AI rename tools send your files to a language model and hope for the best. Gist takes a different approach.

An eight-step deterministic pipeline handles the heavy lifting: extracting dates from financial document headers, identifying organizations using Natural Language Processing, detecting document types from PDF metadata and content structure. The on-device Apple Intelligence model is called only as a last resort — when code-side analysis produces a low-confidence result.

The result: consistent, sortable filenames you can trust.

Date

Scored extraction from document text. "Statement date" beats "due date." Images never get dates — OCR dates are unreliable.

Organization

Header-first detection for financial documents. NLP entity extraction from the first few lines only. No merchant names from transaction tables.

Subject

Eight-step cascade from PDF title to image labels. Code handles the common cases. Apple Intelligence handles the edge cases.
